== Description ==

This simple plugin enhances the default WordPress admin bar by adding custom menu that includes a search field for searching the Codex, and search field for searching the plugin repository. Links to common areas in wp-admin (posts, pages, plugins, media, settings) are included in the drop down menu when you are not in the WordPress admin area.

The plugin also includes a dashboard widget that conveniently displays info about the currently active theme.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `admin-bar-enhancements` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Admin bar should be updated with new menu.
